[TASK] 
Write a dataflow Verilog module for a 4-bit array multiplier. 
 
[MODULE INTERFACE] 
Module name : array_multiplier_4bit 
Inputs      : a [3:0], b [3:0] 
Output      : product [7:0] 

 
[SIGNAL NAMING CONVENTION] 
Partial products  : pp_i_j  (row i, column j) 
                    e.g. pp_0_0 = a[0] & b[0] 
Intermediate sums : s_row_col 
                    e.g. s_1_2 
Carry signals     : c_row_col 
                    e.g. c_1_2 
 
[DATAFLOW RULES] 
- Use only wire declarations and assign statements 
- No module instantiations 
- No always blocks or behavioral constructs 
- No * operator or built-in arithmetic 
- Every signal on the left of assign must be declared as wire 
 
[OUTPUT FORMAT] 
Section 1: All 16 partial product wire declarations + assigns 
Section 2: Row 1 adder expressions (assign statements) 
Section 3: Row 2 adder expressions 
Section 4: Row 3 adder expressions 
Section 5: Final product bit assignments
